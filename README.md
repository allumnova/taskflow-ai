TaskFlow-AI is a high-performance, responsive task management dashboard that leverages real-time cloud synchronization. It features an "AI Priority" algorithm that computes task urgency based on impact and effort metrics.

## Features
- **Cloud Sync**: Powered by Google Firebase (Firestore) for real-time updates across all devices.
- **Smart Prioritization**: Automatically calculates a priority score using the formula `(Impact * 2) - Effort`.
- **First-Class Mobile UI**: Fully responsive design built with **Tailwind CSS**.
- **Allumnova Branding**: Vibrant orange and navy theme with a premium glassmorphic feel.
- **Persistence**: Your tasks are saved securely in the cloud.

## Tech Stack
- **Frontend**: HTML5, Tailwind CSS
- **Backend**: Firebase Firestore (Real-time DB)
- **Deployment**: GitHub Pages

## How to use
1.  Enter your task title.
2.  Select the **Impact** (how important it is) and **Effort** (how much time it takes).
3.  Click **Add Task to Cloud**.
4.  The system will instantly calculate the priority and sync it to the cloud.
